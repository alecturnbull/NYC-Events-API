Welcome to the Why Don't We Street Events API

Getting started

Here’s how to get rolling:

  1. Query /search.json to get a list of all street events in New York

  2. Add parameters start_date and end_date to filter by date

    eg. /posts.json?start_date=today&end_date=tomorrow

    or /posts.json?start_date='6/16/2012'&end_date='7/1/2012'

  3. Add borough to filter by...well, borough

    eg. /posts.json?start_date=today&end_date=tomorrow&borough=manhattan

  4. The JSON comes back with the following keys:

    • Name
    • Address (shortened to first street)
    • Category
    • Description (full address)
    • Borough
    • Start Date
    • End Date
